# Pekara And Pecenjara Permissions

Додадени се две нови привилегии по корисник и локација:

- `CanUsePekara`
- `CanUsePecenjara`

UI:

- во `Корисници` страницата се прикажани како две големи кочки
- администратор одредува дали корисникот ќе внесува за `Пекара`
- администратор одредува дали корисникот ќе внесува за `Печењара`
