# User Location Permissions Module

Имплементирано:

- users list endpoint
- get/update user location permissions endpoints
- admin UI за assignment на права по локација

Права по локација:

- `CanPlan`
- `CanBake`
- `CanRecordWaste`
- `CanViewReports`
- `CanApprovePlan`

Правило:

- корисникот внесува само за своите дозволени локации
